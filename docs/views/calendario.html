BOSS

  • Nombre: Nyx Avatar
  • Nivel Recomendado: 75

Pues bien, este es el jefe final, y tiene nada menos que catorce (CATORCE) formas diferentes que se interponen entre tú y el final. Aunque cada forma no tiene muchos puntos de vida, se van sumando, lo que hace que esta sea una de las batallas más largas del juego. No te sorprendas si tienes que usar un Precious Egg o un Soma o dos mientras te abres paso a través de sus diferentes formas. Una cosa que debería mencionar aquí es que todas las formas de Nyx Avatar tendrán dos turnos seguidos.

Como mínimo, recomendaría tener a los Horus y Rafael Personae para esta batalla. Ambos Personae aprenderán Mediarahan y no tienen debilidades elementales. Un Horus subido de nivel es inmune tanto a la luz como a la oscuridad, y Rafael viene con Null Charm, que puede ahorrarte muchos dolores de cabeza. Si tienes un Tome of the Void (objeto de corazón de Abbadon) puedes equiparlo y no preocuparte por Null Charm o Null Rage. También es recomendable tener un Persona que conozca a Amrita.

  • Anula: Light/Dark
  • Resiste: Todo excepto Almighty

Vale, ahora va en serio. No sólo esta forma tiene cuatro veces más HP que las formas anteriores, sino que también recibe menos daño de cualquier cosa que le lances.

Esta forma ataca principalmente con magia elemental, y puede utilizar habilidades de ruptura para borrar cualquier inmunidad que puedas tener. También puede usar una habilidad llamada Moonless Gown que repele todos los ataques, incluido Almighty, y no se rompe después de un solo golpe como lo hace Tetrakarn/Makarakarn. Si ella prepara el Moonless Gown tu única opción es poner las tácticas de todos en Stand By y esperar hasta que ella lo quite. Mientras el Protagonista pueda evitar ser encantado/enfurecido deberías ser capaz de manejar Night Queen. Si tienes el set de Raphael y el Ring of Serenity equipado, podrás evitar ser encantado/enfurecido, entonces podrás concentrarte en sanar y curar males mientras los miembros de tu grupo acaban con Nyx.
